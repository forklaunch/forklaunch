## Overview

These examples show how to build real applications with ForkLaunch. Each includes a step-by-step guide, the full source code on GitHub, and the AI prompts used to scaffold it.

## Available Examples

### [Dice Roll Node App](/docs/examples/dice-roll-node-app.md)

A dice rolling API built with Node.js, Express, and PostgreSQL. Covers service initialization, database persistence, API endpoints, observability, and local development.

**What it demonstrates:**
- Initializing an application and service with the ForkLaunch CLI
- Defining schemas, entities, and mappers
- Running migrations and local development with Docker Compose
- Built-in OpenTelemetry observability

**GitHub:** [https://github.com/srtandon/dicey-roll](https://github.com/srtandon/dicey-roll)
